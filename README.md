## Travel Web App
A React travel web application

### Tools Used
- React
- node-sass
- eslint
- enzyme
- scrollreveal
- swiper

### Setup Requirements
- node at least v10
- npm
- Your favourite browser

### Getting Started
- Clone this repo
- Install the dependencies, `npm install`
- start the development server, `npm start`
